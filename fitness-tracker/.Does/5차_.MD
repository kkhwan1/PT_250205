# 5차 진행사항 정리

## 1. 포트 충돌 문제 해결
- 포트 3000 사용 중 문제 발생
- netstat 명령어로 프로세스 확인 (PID: 33256)
- taskkill 명령어로 해당 프로세스 종료
- 개발 서버 포트를 3001로 변경하여 충돌 방지
